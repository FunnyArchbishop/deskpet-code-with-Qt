# DeskPet Qt — 古明地恋桌面宠物

基于 **Qt6** 的桌宠应用，以东方 Project 角色**古明地恋**为主题。纯 Qt 库，无原生 API 依赖。

## 特性

- 透明置顶无边框窗口，始终可见
- 支持 **PNG 多帧精灵** 和 **GIF 动画** 两种模式，右键切换
- 动画状态：空闲（眨眼）、行走、坐下、特殊触发
- **快速点击**显示好感度对话气泡 / **按住拖拽**移动位置
- 右键菜单可调整大小（100 / 150 / 200 / 300 / 400）
- 好感度系统：每分钟自动 +5，20/40/60/80/100 解锁古明地恋专属对话
- 自动随机行走 + 屏幕边界弹跳

## 快速开始

### 前置要求
- CMake 3.16+
- Qt6 (Widgets 模块)
- GCC / MSVC (C++17)

### 构建
```bash
cmake -B build -G Ninja -DCMAKE_PREFIX_PATH="<Qt6路径>" -DCMAKE_BUILD_TYPE=Release
cmake --build build
```

### 运行
```bash
./build/deskpet-qt          # Linux / macOS
build\deskpet-qt.exe        # Windows
```

## 操作方式

| 操作 | 效果 |
|---|---|
| **快速点击** | 显示好感度对话气泡（3秒消失） |
| **按住拖拽** | 移动宠物位置 |
| **右键** | 菜单：模式切换 / 调整大小 / 特殊动画 / 退出 |

## 双模式系统

| 模式 | 素材 | 说明 |
|---|---|---|
| 图片模式 | `sprite_*.png` | 多帧 PNG 精灵，交替眨眼/走路 |
| GIF 模式 | `stand.gif` + `creep.gif` | 两张 GIF 驱动全部动画 |

## 好感度对话

| 好感度 | 恋恋的台词 |
|---|---|
| 每加20都会变化 `随着时间推移会逐渐增加哦` | 精彩亟待解锁喵 `买个关子嘻嘻` |

## 自定义素材

替换 `assets/` 下同名文件即可：

| 文件 | 用途 |
|---|---|
| `sprite_idle.png/.gif` | 站立/空闲 |
| `sprite_blink.png/.gif` | 眨眼 |
| `sprite_walk1.png/.gif` | 走路帧1 |
| `sprite_walk2.png/.gif` | 走路帧2 |
| `sprite_sit.png/.gif` | 坐下 |
| `stand.gif` | GIF模式站立 |
| `creep.gif` | GIF模式行走 |
| `triggers/*.png/.gif` | 右键触发特效 |

- `.gif` 优先于 `.png` 加载
- 推荐 128×128 以上带透明通道的素材
- 程序自动缩放适配窗口

## 项目结构

```
deskpet-code-with-Qt/
├── CMakeLists.txt           # CMake 构建配置
├── main.cpp                 # 程序入口
├── petwindow.h/cpp          # 宠物窗口：透明渲染 + 动画状态机 + 交互
├── pet_state.h              # 配置常量 + 状态枚举 + 好感度对话
├── sprite_loader.h/cpp      # 精灵加载器：PNG/GIF → QPixmap/QMovie
├── generate_sprites.py      # 默认精灵生成脚本（可选）
└── assets/                  # 精灵素材（替换此目录自定义角色）
```
